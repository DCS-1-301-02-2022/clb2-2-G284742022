#コンピューターの基礎から高度なプログラミングまで。
高度情報化が進展している現代社会においての、ITを利活用できる知識や技能を学びます。情報工学科では、**基礎スキルから高度なスキルを段階的に学べる教育**、進路を見据えた**幅広いコース選択**によって着実な知識を身につけます。
#３つのコース
１システムエンジニアコース
２インターネット＆セキュリティーコース
３知能情報コース
